# Mutual Fund Analytics Platform

## Project Overview

This project analyzes Indian mutual fund data using Python, SQLite, SQL, and Power BI. The objective is to build a complete ETL pipeline, perform fund performance analysis, calculate risk metrics, and create interactive dashboards.

## Tech Stack

* Python
* Pandas
* SQLite
* SQLAlchemy
* Git & GitHub
* Power BI

## Datasets

* Fund Master
* NAV History
* AUM by Fund House
* Monthly SIP Inflows
* Category Inflows
* Industry Folio Count
* Scheme Performance
* Investor Transactions
* Portfolio Holdings
* Benchmark Indices
